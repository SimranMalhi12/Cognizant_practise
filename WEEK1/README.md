# Mandatory Hands-On Exercises

A collection of Java programming exercises covering core software engineering concepts.

---

## Modules

| # | Module | Path | Topic |
|---|--------|------|-------|
| 1 | JUnit Testing | `JUnit/SettingUpJUnit/` | Unit testing basics with JUnit 5 |
| 2 | Logging | `LoggingExample/` | Log4j2 logging with Spring Boot |
| 3 | Mockito | `Mockito/MockingAndStubbing/` | Mocking and stubbing for unit tests |
| 4 | Factory Method Pattern | `design_pattern/FactoryMethodPatternExample/` | Factory Method design pattern |
| 5 | Singleton Pattern | `design_pattern/SingletonPatternExample/` | Singleton design pattern |
| 6 | Binary Search (DSA) | `dsa/EcommerceSearchFunction/` | E-commerce product search |
| 7 | Recursion (DSA) | `dsa/FinancialForecasting/` | Financial forecasting using recursion |
| 8 | PL/SQL | `pl-sql/` | PL/SQL exercise screenshots |

---

## Tech Used

- Java 17
- Maven
- Spring Boot
- JUnit 5
- Mockito
- Log4j2
- PL/SQL

---

## How to Run

```bash
# Clone
git clone https://github.com/SimranMalhi12/Cognizant_practise.git
cd Cognizant_practise

# Run tests (JUnit / Mockito)
cd JUnit/SettingUpJUnit
mvn test

# Run Spring Boot app (Logging)
cd LoggingExample
mvn spring-boot:run
```
